# CONFIG DOCKER AUTOMATED BUILD WITH GITHUB
## 前提：
     有些镜像可能因为墙的问题，不能访问，那我们需要，但是又下载不了，这时候就很坑爹了。针对这种情况，我们有如下两种方案解决：
     1.通过代理下载，我们可以安装squid代理（不熟悉，不研究）
     2.借助github下载
     下面我们详细说下第二种的实现方法

注：首先你得有github和docker hub的ID
